摘要 
-
优秀的代码结构经受得起版本的迭代与需求的变更<br/>

本项目会总结Java编程中常用的设计模式、结构思想等以供学习参考<br/>

层级包结构 
-
 
+ **exe** ........................................执行用例包 
    + sample ...........................用例包
+ **pattern** ..................................设计模式模型包
    + [pattern name] ................设计模式包
        + [sub pattern name] ..子模式包
+ **storage** .................................模型用例包
    + configuration ...................配置包
    + entity ...............................实体包
    + series ..............................系列包
    + utils .................................工具包
+ **doubt** .....................................存疑结构包
    + [sub doubt] ......................子包
+ **flash** .......................................优秀结构包
    + [sub flash] ........................子包

文例 
-

+ LIST.md ---------快速索引列表

+ MYPOINTS.md-------开发者对于该设计思想的观点与实际应用场景分析
